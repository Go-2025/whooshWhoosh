<h1 align="center">附魔逸响<br></h1>

<p align="center">你好呀！欢迎使用 <b>附魔逸响</b>(WhooshWhoosh) 模组，他提供了更多有趣的附魔，帮助我们战斗和建造。</p>
<h4 align="center">目前模组已添加以下附魔：</h4>
<table align="center">
  <tr>
    <th align="center" width="20%"><b>附魔名称</b></th>
    <th align="center" width="80%"><b>描述</b></th>
  </tr>
  <tr>
    <td align="center"><b>神化</b></td>
    <td align="left">将物品上的所有附魔升至满级</td>
  </tr>
  <tr>
    <td align="center"><b>击飞</b></td>
    <td align="left">将目标击飞</td>
  </tr>
  <tr>
    <td align="center"><b>斩杀</b></td>
    <td align="left">对低血量的敌人造成足以秒杀的伤害</td>
  </tr>
  <tr>
    <td align="center"><b>祭品</b></td>
    <td align="left">使物品变得相当脆弱，并在物品损坏时散落一些经验</td>
  </tr>
  <tr>
    <td align="center"><b>共生</b></td>
    <td align="left">使背包内带有此附魔的物品互相共享耐久值</td>
  </tr>
  <tr>
    <td align="center"><b>涂毒</b></td>
    <td align="left">为目标追加几秒中毒效果</td>
  </tr>
  <tr>
    <td align="center"><b>催化</b></td>
    <td align="left">使受击者现有效果等级提升一级，并减少效果持续时间</td>
  </tr>
  <tr>
    <td align="center"><b>削靡</b></td>
    <td align="left">攻击有概率为目标施加虚弱效果</td>
  </tr>
  <tr>
    <td align="center"><b>碎骨</b></td>
    <td align="left">造成暴击时给予目标虚弱、缓慢和挖掘疲劳的效果</td>
  </tr>
  <tr>
    <td align="center"><b>苍翼</b></td>
    <td align="left">穿戴者能在空中进行二段跳</td>
  </tr>
  <tr>
    <td align="center"><b>腾跃</b></td>
    <td align="left">让取消潜行后的几秒会跳得更高</td>
  </tr>
  <tr>
    <td align="center"><b>法术箭</b></td>
    <td align="left">使箭矢造成法术伤害</td>
  </tr>
  <tr>
    <td align="center"><b>标记</b></td>
    <td align="left">标记目标，射击该目标造成更多伤害</td>
  </tr>
  <tr>
    <td align="center"><b>祭血</b></td>
    <td align="left">消耗血量造成额外伤害</td>
  </tr>
  <tr>
    <td align="center"><b>弹性</b></td>
    <td align="left">被攻击时击退攻击者</td>
  </tr>
</table>
<p align="center" style="color:#BF7B66">目前模组仍在开发阶段，敬请期待！</p>
